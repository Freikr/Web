CSS — Hojas de estilo en cascada

CSS se usa para aplicar estilos a las páginas HTML: colores, tipografías, disposición (layout), etc.

Ejemplo de propiedades

PropiedadDescripción
colorColor del texto
background-colorColor de fondo
displayModo de visualización (block, inline, grid, flex)

Ejemplo: regla CSS

h1 { font-size: 2rem; color: #333; }
