¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web.

Etiquetas semánticas más usadas

  • <header>, <nav>, <main>, <article>, <section>, <footer>
  • <h1>…<h6>, <p>, <a>, <img>, <table>

Ejemplo de tabla (soporte documental)

Comparativa etiquetas
EtiquetaUso
<header>Cabecera de página o sección
<nav>Enlaces de navegación
<article>Contenido independiente
